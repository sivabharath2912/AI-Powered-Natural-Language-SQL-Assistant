1. What AI Helped With

AI was used as a development assistant throughout the project. It helped with:

* Understanding the project requirements and designing the overall architecture.
* Generating Python code for individual modules.
* Building the Natural Language to SQL conversion workflow.
* Implementing query validation and execution logic.
* Creating the Streamlit user interface.
* Adding data visualization features and CSV export functionality.
* Assisting in debugging and fixing errors.
* Preparing documentation and test cases.

2. What AI Got Wrong

During development, some AI-generated responses required manual corrections. Examples include:

* Incorrect import paths between modules.
* Outdated model names and API configurations.
* Repeated display of SQL statements in the Streamlit interface.
* Session state issues causing unexpected behavior after button clicks.
* Improper placement of sidebar components and query history.
* Missing columns in generated INSERT statements.
* Indentation and control flow errors that required manual debugging.

These issues were identified, corrected, and tested before integration.
